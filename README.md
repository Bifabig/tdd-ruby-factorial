# 📖 TDD With RSpec Project <a name="about-project"></a>

**TDD With RSpec Project:** A Ruby class with utility methods for factorial calculation, string reversal, and FizzBuzz evaluation. Developed using Test-Driven Development (TDD) principles, the class offers reliable and tested functionalities for common operations in Ruby projects.

## 🛠 Built With <a name="built-with"></a>

  <ul>
    <li>Ruby</li>
    <li>RSpec</li>
    <li>TDD</li>
  </ul>

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Factorial**
- **Reverse**
- **FizzBuzz**
